# Poeiria

Poeiria é um projeto de software como **desenvolvedor Full Stack** para mostrar **poemas** de diversos escritores e permitir a publicação de poemas autorais.  
É uma forma simples de registrar ideias e compartilhá-las.  
Este é um projeto pessoal, apenas para uso próprio.

![Tela inicial](./doc/assets/poeiria.png)

Divirta-se e desenvolva sua criatividade, treine sua escrita e aprenda com diferentes pontos de vista e estilos de escrita.
